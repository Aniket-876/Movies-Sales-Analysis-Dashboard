IMDb Movie Sales & Trends Analysis
A comprehensive data visualization project focused on analyzing revenue, ratings, and performance metrics of global movies (2006-2016).

🚀 Key Features
Revenue Analysis: Tracked gross earnings and ROI across different genres.

Rating Correlations: Analyzed the relationship between IMDb scores and box office success.

Temporal Trends: Year-over-year growth and seasonal release patterns.

Interactive Filters: Dynamic slicers for Genre, Year, and Director-specific insights.

🛠️ Tech Stack
Data Tool: Power BI / Excel

Language: Python (Pandas) for Data Cleaning

Dataset: IMDb Top 1000 Movies (2006-2016)

💡 Key Insight
Identified that high-budget movies don't always correlate with high ratings, but specific genres (like Sci-Fi and Adventure) show a 40% higher revenue consistency.
